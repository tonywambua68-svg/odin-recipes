# Odin Recipes

This is a simple recipe website project created as part of The Odin Project curriculum.

## Skills demonstrated
- Building a basic HTML website with multiple pages
- Using Git for version control (commits, branching, pushing to GitHub)
- Writing clean and semantic HTML
- Organizing project files

## Reflection
By completing this project, I will have practiced structuring an HTML project, linking pages, and using Git to manage my code. I will also have learned how to write meaningful commit messages and push my work to a remote repository on GitHub.
